# Autorendi veebileht

## Projekti kirjeldus
See projekt on autorendi veebileht, kus kasutajad saavad registreeruda, sisse logida ja autosid rentida. Administraator saab hallata autosid ja broneeringuid.

Projekt töötab Dockeriga, seega saab seda lihtsalt käivitada igas arvutis käsuga `docker-compose up`.

---

## Funktsioonid

### Kasutaja
- Kasutaja saab registreeruda
- Kasutaja saab sisse logida
- Kasutaja saab vaadata autosid
- Kasutaja saab autot rentida
- Sama autot ei saa samaks ajaks mitu korda rentida

### Admin
- Admin saab autosid lisada
- Admin saab autosid muuta
- Admin saab autosid kustutada
- Admin saab vaadata broneeringuid
- Admin saab kustutada broneeringuid
